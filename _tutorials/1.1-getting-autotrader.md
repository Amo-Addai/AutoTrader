To get AutoTrader, clone the [Github repository](https://github.com/kieran-mackle/AutoTrader) onto your machine 
and install the required dependencies.

```
$ git clone https://github.com/kieran-mackle/AutoTrader
```

## Dependencies
- NumPy
- TaLib
- pandas
- pytz
- datetime
- yaml
